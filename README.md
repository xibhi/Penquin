# Installation

Run this single command to install the full toolkit:

```bash
wget https://raw.githubusercontent.com/xibhi/Penquin/refs/heads/master/ToolsInstaller.sh && sudo bash ToolsInstaller.sh
```

This pulls down and installs every essential tool used in professional bug bounty hunting. Not random utilities — the ones that actually matter.

> Requires a Debian/Ubuntu-based Linux distro. Make sure you have `wget` and `bash` available.

---

# What's Inside

**Browser Tools** — Security extensions for passive scanning, request tampering, and vulnerability detection, ready to go out of the box.

**Structured Recon** — Subdomain enumeration → endpoint mapping → targeted scans. A clear path from zero to findings.

**Pre-Built Commands** — Scanning, exploitation, and automation commands you can run immediately. Stop retyping the same flags from memory.

**Hacker Insights** — Writeups, YouTube channels, Twitter follows, books, CLI tips, WSL setup guides, and more. Everything you need to keep leveling up.

**Android Bug Bounty** — A full Android pentesting section covering Frida, Drozer, Objection, automated scanners, intentionally vulnerable apps, and real bug reports to learn from.
